color: sirve para elgir color de texto

font-family: elgir tipografia de texto
* Si la tipografia esta compuesta de dos palabras va en comillas ''
* Por si el navegador no carga alguna tipografia se puede poner una o varias tipografias separadas por coma para que cargue la siguiente en orden

font-size: modifica el tamano de la letra por UNIDADES DE MEDIDA (px, rem, %)
* Por defecto el navegador pone la letra el 16px

font-weight: modifica el grosor de letra (del grosor 100 al grosor 900 o en bold o bolder)

* Bold: va del grosor por defecto al siguiente grosor
* Bolder: si estas en un grosor intermedio(400) Bolder buscara el siguiente grosor (600)
