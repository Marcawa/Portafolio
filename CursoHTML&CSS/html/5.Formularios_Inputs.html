input: Almacena datos * type: tipo de input text : ingresar texto color : ingresar color email : ingresar email y obliga a poner "@"" y el resto sin ".com" password : ingresar contrasena tapada con "*****************" checkbox : marcar y desmarcar varios CHECKBOX text : marcar solo un RADIO sin poder desmarcar file : ingresar un archivo date : ingresar una fecha submit : boton de enviar al form * value : valor del input (nombre, numero, etc..) * required : requiere que se llene el input * name : identificador del input para el servidor * placeholder : texto que desaparece al escribir en input
